# Razorpay Payment Gateway Backend

  Razorpay Payment Gateway Backend

## Table of Contents

- [Usage](#usage)
- [Configuration](#configuration)


## Usage

1. Install packages:

   ```bash
   npm install
   ```
   
2. Start the server:

    ```bash
   node app.js
   ```

## Configuration

1.  Create a .env file in the root directory and add your configuration:

```
KEY_ID=KEY_ID
KEY_SECRET=KEY_SECRET
CURRENCY_PRICE=100
CURRENCY_CODE=INR
PORT=8080


```
